1 Introduction

This is a physics game where you as a black whole float around in space and
eat planets and other objects.

1.1 Purpose of application

This project aims to create a physics based game that provides
entertainment to the user.

1.2 General characteristics of application

The game is physics based android game, controllable to touch
controllers. It is single player.

The player moves the black hole eating swallowing objects until he
dies. There are no time constraints. If the black hole dies it is respawned
and a life is used up. This goes on until the player has run out
lives, then it's game over. The black hole dies when it is eaten by an
other black hole.

1.3 Scope of application

IT is single player, but the enemy black holes are controlled by an
AI.

1.4 Objectives and success criteria of the project

  1. It should be possible to play the game for as long as the player
  is still alive. Stages are generated to keep the game going.
